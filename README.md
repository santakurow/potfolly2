# Potfolly

ポートフォリオ、技術共有サイト

## 仕様

### フレームワーク
- Ruby on Rails6 (サーバーサイド)
- React.js (フロントエンド)

### ライブラリ
- Bootstrap
- material-ui
- jQuery

### DB
- MySQL
- postgresql

### モジュールバンドラー
- Webpack

### テスト
- RSpec

### デプロイ
- heroku (本体)
- AWS S3 (画像データ)
